## 一、官方 401 表結構（財政部規範）

### 401 表全名
**營業人銷售額與稅額申報書**（適用於：一般稅額計算，專營應稅營業人）

### 申報週期
- 每 **2 個月** 申報一次（共 6 期）
- 期別：1-2月、3-4月、5-6月、7-8月、9-10月、11-12月
- 於次期開始 **15 日前** 完成申報（如 1-2 月份在 3/15 前申報）

### 401 表三大區塊

#### 第一區：銷項（Sales / Output）

| 行號 | 欄位名稱 | 說明 |
|------|---------|------|
| 1 | 應稅銷售額（稅率 5%）— 銷售額 | 開給買方的含稅發票，拆出未稅金額 |
| 2 | 應稅銷售額（稅率 5%）— 稅額 | = 銷售額 × 5% |
| 3 | 零稅率銷售額（合計） | 外銷或特定勞務，稅額為 0 |
| 3a | └ 外銷貨物 | 實體貨物出口 |
| 3b | └ 與外銷有關之勞務 | 國內提供、國外使用之勞務 |
| 3c | └ 依法零稅率之銷售額 | 其他法定零稅率項目 |
| 4 | 免稅銷售額 | 依營業稅法第 8 條免稅項目 |
| 5 | 銷項合計銷售額 | = 行1 + 行3 + 行4 |
| 6 | 銷項合計稅額 | = 行2（僅應稅部分有稅額） |

#### 第二區：進項（Purchases / Input）

| 行號 | 欄位名稱 | 說明 |
|------|---------|------|
| 7 | 統一發票扣抵聯 — 進貨及費用（筆數） | 三聯式、電子發票等 |
| 8 | 統一發票扣抵聯 — 進貨及費用（金額） | 未稅金額 |
| 9 | 統一發票扣抵聯 — 進貨及費用（稅額） | 可扣抵進項稅額 |
| 10 | 統一發票扣抵聯 — 固定資產（筆數） | 購入機器設備等 |
| 11 | 統一發票扣抵聯 — 固定資產（金額） | 未稅金額 |
| 12 | 統一發票扣抵聯 — 固定資產（稅額） | 可扣抵進項稅額 |
| 13 | 三聯式收銀機發票、載有稅額之其他憑證（筆數） | |
| 14 | 三聯式收銀機發票、載有稅額之其他憑證（金額） | |
| 15 | 三聯式收銀機發票、載有稅額之其他憑證（稅額） | |
| 16 | 海關代徵營業稅繳納證（筆數） | 進口貨物 |
| 17 | 海關代徵營業稅繳納證（金額） | |
| 18 | 海關代徵營業稅繳納證（稅額） | |
| 19 | 進項合計（筆數） | |
| 20 | 進項合計（金額） | = 行8+11+14+17 |
| 21 | 進項合計（稅額） | = 行9+12+15+18 |

#### 第三區：稅額計算

| 行號 | 欄位名稱 | 計算公式 |
|------|---------|---------|
| 22 | 銷項稅額 | = 行6 |
| 23 | 進項稅額 | = 行21 |
| 24 | 上期（月）累積留抵稅額 | 從上期結轉 |
| 25 | 不得扣抵比率 | 兼營免稅或非應稅時才填 |
| 26 | 不得扣抵之進項稅額 | = (行23 + 行24) × 行25 |
| 27 | 本期（月）應實繳稅額 | = 行22 - 行23 - 行24 + 行26（正數繳稅） |
| 28 | 本期（月）申報留抵稅額 | 若行27為負數，轉為留抵 |
| 29 | 本期零稅率溢付退稅 | 零稅率產生的退稅金額 |

### 不得扣抵進項稅額的項目（營業稅法第 19 條）

1. 購買自用乘人小汽車
2. 交際應酬用之貨物或勞務
3. 酬勞員工個人之貨物或勞務
4. 非供本業及附屬業務使用之貨物或勞務

### 401 vs 403 vs 405 表差異

| 特徵 | 401 | 403 | 405 |
|------|-----|-----|-----|
| 適用對象 | 專營應稅營業人 | 兼營應稅+免稅/特種稅額 | 小規模營業人 |
| 申報頻率 | 每 2 個月 | 每 2 個月 | 每 3 個月 |
| 免稅銷售額欄位 | 有（簡單） | 有（詳細） | 無 |
| 不得扣抵比率計算 | 簡單 | 複雜（比例扣抵） | 無 |
| 稅率 | 5% | 5% + 特種稅率 | 1%~5% |

| 缺口 | 嚴重度 | 說明 |
|------|--------|------|

| 進貨退出或折讓 | MEDIUM | 媒體檔有格式代號 23/24/33/34，但系統未處理退出折讓 |
| 海關代徵營業稅 | MEDIUM | 進口貨物的營業稅，媒體檔有格式 28/29 但未使用 |

| 申報期限提醒 | LOW | 無 cron 提醒次期 15 日前需申報 |

| 購買國外勞務稅額 | LOW | 特殊項目，多數中小企業不適用 |
| 進項憑證類型細分 | LOW | 三聯式/收銀機/海關繳納證，目前統一為電子發票格式 |

### 計算公式對比

```
官方公式：
應納稅額 = 銷項稅額 - (進項稅額 + 上期留抵) + 不得扣抵之進項稅額

系統公式（目前）：
netTax = outputTax - inputTax
// 缺少：上期留抵、不得扣抵比率計算
```

## 三、系統操作教學

### Step 1：進入營業稅申報頁面

路徑：**會計模組 > 稅務報表**

對應路由：`/accounting/reports`

頁面元件：`app/accounting/reports/TaxReportDashboard.tsx`

### Step 2：選擇申報期間

1. 選擇「年度」（預設為當前年度，可選最近 5 年）
2. 選擇「期別」（1-2月 至 11-12月）
   - 系統預設上一期（例如 3 月時預設 1-2 月）
3. 點擊「產生報表」按鈕

> 系統會呼叫 API：`GET /api/accounting/reports/tax?form=401&year=2026&bi_month=1`

### Step 3：查看摘要

「摘要」分頁顯示三大區塊：

1. **稅額總覽卡片**
   - 藍色：銷項稅額 + 發票張數
   - 綠色：進項稅額 + 發票張數
   - 紅色/橙色：應納稅額 或 應退稅額

2. **銷項結構表**（附圓餅圖）
   - 應稅 5%：金額、稅額
   - 零稅率：金額（稅額為 0）
   - 免稅：金額（稅額為 0）

3. **進項結構表**（附圓餅圖）
   - 可扣抵：金額、稅額
   - 不可扣抵：金額、稅額

### Step 4：檢查銷項明細

切換到「銷項明細」分頁，可查看：
- 發票號碼、日期
- 交易對象名稱、統一編號
- 未稅金額、稅額
- 稅別標籤（應稅 5% / 零稅率 / 免稅）

### Step 5：檢查進項明細

切換到「進項明細」分頁，可查看：
- 發票號碼、日期
- 交易對象名稱、統一編號
- 未稅金額、稅額
- 扣抵性標籤（可扣抵 / 不可扣抵）

### Step 6：預覽 XML

切換到「XML 預覽」分頁，可查看即將匯出的 XML 內容。

XML 結構：
```xml
<VAT401>
  <Header>年度、期別、統編、公司名稱</Header>
  <Sales>
    <Taxable>應稅銷售（筆數、金額、稅額）</Taxable>
    <ZeroRated>零稅率（筆數、金額）</ZeroRated>
    <Exempt>免稅（筆數、金額）</Exempt>
  </Sales>
  <Purchases>
    <Deductible>可扣抵（筆數、金額、稅額）</Deductible>
    <NonDeductible>不可扣抵（筆數、金額、稅額）</NonDeductible>
  </Purchases>
  <TaxCalculation>
    銷項稅額、進項稅額、應納稅額、是否退稅
  </TaxCalculation>
</VAT401>
```

### Step 7：匯出檔案

兩種匯出格式：

1. **下載 XML** — 申報書 XML 格式
   - 檔名：`VAT401_年度_期別.xml`

2. **下載媒體檔** — 財政部離線建檔系統格式
   - 檔名：`統編.TXT`
   - 每筆資料 81 bytes 固定長度
   - 包含所有進銷項發票明細
   - 可直接匯入財政部「營業稅電子申報繳稅系統」

### 媒體檔欄位結構（81 Bytes）

```
| 位置  | 長度 | 欄位名稱       | 說明                        |
|-------|------|---------------|----------------------------|
| 1-2   | 2    | 格式代號       | 25=進項電子發票, 35=銷項電子發票 |
| 3-11  | 9    | 稅籍編號       | 統編8碼+分支碼1碼            |
| 12-18 | 7    | 流水號         | 0000001 起                  |
| 19-23 | 5    | 資料所屬年月    | 民國年3碼+月2碼              |
| 24-31 | 8    | 買受人統編      | 銷項=買方, 進項=自己          |
| 32-39 | 8    | 銷售人統編      | 銷項=自己, 進項=賣方          |
| 40-49 | 10   | 發票字軌號碼    | 2碼英文+8碼數字              |
| 50-61 | 12   | 銷售額         | 未稅金額，靠右補零            |

| 75    | 1    | 通關方式       | 零稅率專用                   |
| 76-81 | 6    | 保留欄位       | 空白                        |
```

## 四、營業稅計算範例

### 範例：某公司 2026 年 1-2 月

**銷項：**
- 應稅銷售額：$500,000 → 稅額 $25,000
- 零稅率銷售額：$100,000 → 稅額 $0
- 免稅銷售額：$0

**進項：**
- 進貨及費用：$300,000 → 稅額 $15,000（可扣抵）
- 固定資產：$50,000 → 稅額 $2,500（可扣抵）
- 交際費：$20,000 → 稅額 $1,000（不可扣抵）

**稅額計算：**
```
銷項稅額 = $25,000
進項稅額 = $15,000 + $2,500 = $17,500（僅可扣抵部分）
上期留抵 = $0

應納稅額 = $25,000 - $17,500 - $0 = $7,500
→ 本期需繳納營業稅 $7,500
```

**不可扣抵的 $1,000 稅額**不能用來抵扣，因為交際費屬於營業稅法第 19 條規定的不得扣抵項目。

## 五、改善建議優先順序

### P0 — 必須修（影響正確性）

1. **新增「累積留抵稅額」欄位**
   - 在 `Form401Data.taxCalculation` 新增 `carryOverCredit` 欄位
   - 需要資料庫表儲存每期留抵金額
   - 稅額計算公式修正為：`netTax = outputTax - inputTax - carryOverCredit`

2. **進項分離「進貨及費用」vs「固定資產」**
   - 在 `Form401Data.purchases` 新增 `fixedAsset` 分類
   - 發票表新增 `is_fixed_asset` 欄位（或由會計科目判斷）
   - UI 增加固定資產行

### P1 — 應該修（提升實用性）

3. **不得扣抵比率計算**
   - 新增 `nonDeductibleRatio` 欄位
   - 兼營免稅時自動計算比例

4. **零稅率子分類**
   - 區分外銷貨物、勞務、其他

5. **進貨退出/折讓處理**
   - 支援負數發票或折讓單

### P2 — 可以改（增加便利性）

6. 申報期限提醒（cron job）
7. 官方格式 XML 匯出
8. 紙本 401 表列印版面
9. 進項憑證類型細分
