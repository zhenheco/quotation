## 🔧 採集工具選項

### 選項 1：Playwright 自動化

使用 Playwright 自動截取：

```bash
pnpm test:e2e:playwright
```

### 選項 2：手動截取

1. 開啟瀏覽器開發者工具 (F12)
2. 設定視窗大小為 1920x1080
3. 瀏覽到目標頁面
4. 使用截圖工具 (macOS: Cmd+Shift+4)
5. 存放到 `docs/screenshots-tutorial/screenshots/`

### 選項 3：瀏覽器擴充功能

推薦使用：
- **Firefox Screenshots**：內建功能
- **Nimbus Screenshot**：Chrome/Edge 擴充
